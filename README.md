# mae-cifar10-masking

Masked Autoencoder (MAE) experiments on CIFAR-10 dataset.

## Folder Structure

```
mae-cifar10-masking/
├── configs/     # Configuration files for experiments
├── data/        # Dataset storage (CIFAR-10)
├── src/         # Source code (models, training, utilities)
├── scripts/     # Utility and experiment scripts
└── results/     # Experiment outputs (checkpoints, logs, plots)
```

## Getting Started

1. Clone this repository
2. Install dependencies (see requirements)
3. Download CIFAR-10 dataset
4. Run experiments using scripts in `scripts/`
